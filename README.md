# StarShare コーポレートサイト

株式会社スターシェア（StarShare Inc.）のコーポレートサイトです。

## 構成

```
starshare-corp/
├── index.html        … トップページ（1ページ完結）
├── css/style.css     … スタイル
├── js/main.js        … メニュー開閉などの簡単なJS
└── images/           … 画像（ロゴ・実績写真など）
```

## セクション

- 事業内容（Business）
- 実績（Works）
- 会社概要（Company）
- お問い合わせ（Contact / Netlify Forms 対応）

## ホスティング

Netlify でホスティング。`main` ブランチに push すると自動デプロイされます。

## 編集メモ

- 会社概要の「―――」部分は実際の情報に差し替えてください。
- 実績セクションの画像・内容は後ほど差し替えてください。
- ロゴ画像は `images/` に配置し、ヘッダーのテキストロゴと差し替え可能です。
